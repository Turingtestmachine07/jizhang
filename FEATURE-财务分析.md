# 财务分析功能说明

## 新增功能概览

### 1. 财务分析页面 (FinancialAnalysis.vue)

全新的财务分析模块，提供多维度的盈亏分析和经营洞察。

## 核心功能

### 📊 盈亏概览

#### 关键财务指标（8个核心指标卡片）

1. **营业收入** - 总订单金额
   - 显示所选时间段内的所有订单总金额
   - 蓝色主题

2. **实收金额** - 已收款金额
   - 显示实际已到账的金额
   - 绿色主题

3. **总支出** - 成本费用支出
   - 显示所有支出项目的总和
   - 红色主题

4. **毛利润** - 收入 - 支出
   - 公式: 营业收入 - 总支出
   - 盈利显示绿色，亏损显示红色

5. **净利润** - 实收 - 支出
   - 公式: 实收金额 - 总支出
   - 反映真实现金盈利情况

6. **应收款** - 待收款金额
   - 公式: 营业收入 - 实收金额
   - 橙色警示主题

7. **毛利率** - 毛利润/收入
   - 公式: (毛利润 / 营业收入) × 100%
   - 反映定价和成本控制能力

8. **净利率** - 净利润/收入
   - 公式: (净利润 / 营业收入) × 100%
   - 反映真实盈利能力

#### 盈亏趋势分析表

按时间维度展示详细的盈亏数据：
- 营业收入
- 总支出
- 毛利润（带正负标识）
- 净利润（带正负标识）
- 毛利率（带颜色标签）
- 实收金额

支持按**天/月/年**三种粒度统计。

### 💰 成本分析

#### 支出分类统计表

- 显示各个支出分类的详细统计
- 包含支出笔数和总金额
- **可视化占比**：使用进度条展示各分类占总支出的百分比
- 帮助识别主要成本项

### 💵 现金流分析

#### 现金流量表

- **现金流入**：实际收到的款项
- **现金流出**：实际支付的费用
- **净现金流**：流入 - 流出
- **应收未收**：已开单但未收款的金额

> 用于监控企业的现金健康状况

### 🔍 经营洞察

#### 1. 盈利能力分析

- **毛利率分析**
  - ≥ 30%: 盈利能力优秀（绿色）
  - 15% ~ 30%: 盈利能力良好（橙色）
  - < 15%: 需要控制成本（红色）

- **净利率分析**
  - ≥ 20%: 现金获取能力强（绿色）
  - 10% ~ 20%: 现金流健康（橙色）
  - < 10%: 需加强回款（红色）

#### 2. 成本控制分析

- **成本率分析**
  - ≤ 60%: 成本控制优秀（绿色）
  - 60% ~ 80%: 成本控制良好（橙色）
  - > 80%: 成本偏高，需优化（红色）

- **应收款占比分析**
  - ≤ 20%: 回款及时（绿色）
  - 20% ~ 40%: 回款正常（橙色）
  - > 40%: 存在大量应收款（红色）

#### 3. 智能经营建议

系统根据财务数据自动提供经营建议：

**亏损预警**（净利润 < 0）
- 分析主要支出项目，控制非必要开支
- 提高产品定价或优化产品结构
- 加强应收款催收，改善现金流

**利润率偏低**（净利率 < 10%）
- 优化成本结构，寻找降本增效机会
- 提升高毛利产品销售占比
- 改善运营效率，提高周转率

**应收款占比较高**（> 30%）
- 加强客户信用管理，设置合理账期
- 建立催收机制，定期跟进应收款
- 考虑提供现款优惠，促进及时回款

**经营状况良好**（净利率 ≥ 15%）
- 保持当前经营策略，稳中求进
- 考虑扩大市场份额或产品线
- 建立现金储备，应对市场波动

## 筛选功能

### 日期范围筛选
- 支持自定义日期范围
- 默认显示最近30天数据

### 统计粒度
- **按天**：适合短期分析
- **按月**：适合中长期趋势分析
- **按年**：适合年度对比

## 数据导出

支持将完整的财务分析报表导出为 Excel 文件：
- 包含销售统计
- 包含产品统计
- 包含客户统计
- 文件名：`财务分析报表_YYYY-MM-DD.xlsx`

## 使用场景

### 1. 日常经营监控
- 每天/每周查看盈亏概览
- 监控净现金流变化
- 关注应收款情况

### 2. 月度/季度复盘
- 分析月度/季度盈利情况
- 对比不同时期的经营数据
- 识别经营趋势

### 3. 成本优化决策
- 查看成本分析，识别主要支出项
- 分析成本率，找出优化空间
- 控制非必要支出

### 4. 定价策略调整
- 查看毛利率是否达标
- 根据成本率调整产品定价
- 优化产品结构，提升整体利润率

### 5. 现金流管理
- 监控净现金流，确保企业流动性
- 控制应收款占比
- 平衡收支，避免现金断流

## 计算公式说明

```
营业收入 = ΣΣ 所有订单的 total_amount

实收金额 = Σ 所有订单的 paid_amount

总支出 = Σ 所有支出记录的 amount

毛利润 = 营业收入 - 总支出

净利润 = 实收金额 - 总支出

应收款 = 营业收入 - 实收金额

毛利率 = (毛利润 / 营业收入) × 100%

净利率 = (净利润 / 营业收入) × 100%

成本率 = (总支出 / 营业收入) × 100%

应收款占比 = (应收款 / 营业收入) × 100%
```

## 技术实现

### 前端
- 文件位置：`client/src/views/FinancialAnalysis.vue`
- 使用 Vue 3 Composition API
- Element Plus UI 组件
- 响应式设计，支持移动端

### 后端 API
- 使用已有的统计 API：
  - `GET /api/stats/sales` - 获取销售数据
  - `GET /api/stats/expenses` - 获取支出数据
  - `GET /api/stats/expenses/categories` - 获取支出分类统计
  - `GET /api/stats/dashboard` - 获取仪表盘数据
  - `GET /api/stats/export/excel` - 导出报表

### 数据处理
- 前端合并销售和支出数据
- 自动计算各项财务指标
- 智能生成经营建议
